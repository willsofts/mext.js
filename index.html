Moleculer Express Template

This template made of Moleculer Service as API back end engine and Express framework build web application.
Moleculer is a fast, modern and powerful microservices framework for Node.js and Express is a minimal and flexible Node.js web application framework.
Template consist of private components and libraries with the following.
Node.js Modules.
will-utilThis is the utilities modules.
will-sqlThis is the relational database management modules.
will-storeThis is an external storage utility for third party storage library.
will-libThis is the internal library modules.
will-dhThis is the diffie-helman modules.
will-dbThis is the relational database service for moleculer modules.
will-apiThis is web API gateway adapter for moleculer modules.
will-runThis is the runner for moleculer services and express application framework modules.
will-coreThis is the internal core for moleculer modules.
will-servThis is the routing for moleculer and express modules.
Client Modules. This support for build Single Page Application (SPA) with jQuery and Bootstrap.
will-appThis is the program function utilities. This module using bootstrap 4.
will-controlThis is VUE.js controls for vue application.
will-app5This is the program function utilities. This module using bootstrap 5.
Example Vue Application. This single page application consist of parent and child application to create enterprise web application.
vuesureThis is demonstration of using Vue 3 framework and controls create single page application for login page. This reserved for parent application.
vueappThis is demonstration of using Vue 3 framework and controls create single page application. This reserved for child application.
Extended Template
assureThis is the template sources for development.
mextThis is derived template for production.
Training Materials.
simple-sqlThis is demostration of will-sql usage.
simple-dbThis is demostration of will-db usage.
simple-serviceThis is demostration of communication services using bus as a transporter.
simple-reportThis is demostration of PDF and Excel report file usage.
Service in Template.
Service MeThis is how to provided service in template.
